# 脚本工具说明

## crawl_sample.py - GeoGebra 材料下载脚本

### 功能简介

该脚本用于从 GeoGebra 官方 API 获取中文几何教学材料并下载。主要功能包括：

1. **获取材料列表**：从 GeoGebra API 获取几何主题的教学材料
2. **智能下载**：自动跳过已下载的材料，避免重复下载
3. **文件解压**：解压 GGB 文件（ZIP 格式）到指定目录
4. **图片下载**：下载材料的预览图和缩略图

### 技术实现

- **数据源**：GeoGebra API (https://api.geogebra.org/v1.0)
- **文件格式**：GGB (GeoGebra 文件，本质是 ZIP 压缩包)
- **输出**：原始 XML 文件和相关资源

### 使用方法

#### 安装依赖

```bash
pip install requests
```

#### 运行脚本

```bash
cd ./docs/_/scripts 
python crawl_sample.py
```

#### 运行效果

脚本每次运行会自动：
1. 扫描本地已下载的材料（检查 geogebra.xml 文件）
2. 从 API 获取新材料列表（过滤已下载的）
3. 下载新的 20 个材料
4. 解压文件并下载图片

### 输出目录结构

```
docs/sample/
├── materials.json             # 材料列表索引
└── {material_id}/
    ├── geogebra.xml           # GeoGebra 原始 XML 文件
    ├── thumbnail.png          # 缩略图（如果有）
    ├── preview.png            # 预览图（如果有）
    └── 其他解压文件...
```

### materials.json 格式

```json
[
  {
    "id": "material_id",
    "title": "材料标题"
  }
]
```

### 配置说明

脚本中的主要配置参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `BASE_DIR` | `../../sample` | 下载目录路径 |
| `API_BASE` | `https://api.geogebra.org/v1.0` | API 基础地址 |
| `target_count` | `20` | 每次下载的材料数量 |

### 注意事项

1. 部分材料可能没有下载链接，脚本会自动跳过并记录错误
2. 建议定期运行脚本以获取最新材料
3. 下载的材料数量没有限制，每次运行都会继续下载新的材料
4. 脚本会自动检查本地是否已存在 `geogebra.xml` 文件来判断是否已下载
