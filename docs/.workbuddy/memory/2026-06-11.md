## 批量替换 /_/ 路径为 /asset/

- 目标：所有 `.htm` 文件中的 `/_/css/site.css`、`/_/js/site.js` 等路径，将 `/_/` 改为 `/asset/`
- 命令：Python glob 遍历 + re.sub 批量替换
- 结果：修改了 **1376 个文件**，共替换 **3940 处** `/_/` → `/asset/`
- 涵盖 en 和 zh 两个目录下所有深度的页面（depth=2 如 `../../_/`，depth=4 如 `../../../../_/`）
- 注意：`data-ui-root-path="../../_"` 中末尾无斜杠的 `/_` 未被本次替换（不在需求范围内）
