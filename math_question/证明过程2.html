几何证明：BE=AE+CE（方法二）

已知条件

  • △ABC 是等边三角形，A 在最上面，B 在左侧，C 在左下
  • AP 平行于画布纵向边（垂直方向）
  • D 是点 C 关于 AP 的对称点
  • E 是 BD 与 AP 的交点

证明过程

因为 D 是点 C 关于 AP 的对称点，所以 AP 是线段 CD 的垂直平分线。

因此，AC = AD，CE = DE（垂直平分线上的点到线段两端的距离相等）。

又因为 △ABC 是等边三角形，所以 AB = AC = BC。

由 AC = AD 和 AB = AC 可得 AB = AD，即 △ABD 是等腰三角形。

因为 AP 是对称轴，所以 ∠ACE = ∠ADE（对称性质）。

在 BE 上截取 BF = CE，连接 AF。

在 △ABF 和 △ACE 中：

AB = AC （已证）

∠ABF = ∠ACE （因为 ∠ABF = ∠ABD = ∠ADE = ∠ACE）

BF = CE （构造）

所以 △ABF ≌ △ACE（SAS 全等判定）。

因此，AF = AE，∠BAF = ∠CAE。

因为 ∠BAC = 60°，所以 ∠FAE = ∠BAC = 60°（∠FAE = ∠BAC - ∠BAF + ∠CAE = 60° - ∠BAF + ∠BAF = 60°）。

因此，△AFE 是等边三角形（因为 AF = AE，∠FAE = 60°）。

所以，FE = AE。

因此，BE = BF + FE = CE + AE。

证毕。
