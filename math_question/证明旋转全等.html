几何证明：BE=AE+CE

已知条件

  • △ABC 是等边三角形，A 在最上面，B 在左侧，C 在左下
  • AP 平行于画布纵向边（垂直方向）
  • D 是点 C 关于 AP 的对称点
  • E 是 BD 与 AP 的交点

证明过程

因为 D 是点 C 关于 AP 的对称点，所以 AP 是线段 CD 的垂直平分线。

因此，AC = AD，CE = DE（垂直平分线上的点到线段两端的距离相等）。

又因为 △ABC 是等边三角形，所以 AB = AC = BC。

由 AC = AD 和 AB = AC 可得 AB = AD，即 △ABD 是等腰三角形。

现在，我们将 △ACE 绕点 A 逆时针旋转 60°，得到 △ABF。

因为 △ABC 是等边三角形，所以旋转 60° 后，AC 会与 AB 重合。

因此，△ABF ≌ △ACE（旋转全等）。

所以，BF = CE，AF = AE，∠BAF = ∠CAE。

又因为 ∠BAC = 60°，所以 ∠FAE = ∠BAC = 60°。

因此，△AFE 是等边三角形（因为 AF = AE，∠FAE = 60°）。

所以，FE = AE。

现在，我们需要证明点 F 在 BE 上。

因为 ∠ABF = ∠ACE，而 ∠ACE = ∠ADE（因为 AP 是对称轴）。

又因为 △ABD 是等腰三角形，所以 ∠ABD = ∠ADE。

因此，∠ABF = ∠ABD，所以点 F 在 BE 上。

因此，BE = BF + FE = CE + AE。

证毕。
